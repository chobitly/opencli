# Sina Blog Archiving Skill

> [!NOTE]
> **人眼快读 / Human Quick Guide**
>
> ## 全能存储命令：`sinablog save`
>
> - **默认行为**：同时保存 Markdown 笔记和高清图片到指定文件夹。
> - **Obsidian 适配**：使用 `![[./attachments/xxx]]` 语法，支持在 Obsidian 中直接预览图片。
> - **文件名规范**：自动以 `YYYY-MM-DD-标题` 命名，方便按日期排序。
> - **自定义输出与附件文件夹**：支持 `--output` 和 `--attachments` 参数。
> - **以上规范 `save` 和 `batch-save` 命令均完全一致。**
>
> ## 快速示例
>
> - **生成筛选清单**：
>   - `opencli sinablog user-collection <UID> --output list.md`
> - **批量采集已标记项**：
>   - `opencli sinablog batch-save list.md`
> - **单篇文章采集**：
>   - `opencli sinablog save "URL"`

## Core Capabilities

### 1. Full Article Archiving (`save`)

The primary tool for archiving. It creates a Markdown file with full metadata and downloads all associated images into a local folder.

**Command:**

```bash
opencli sinablog save "<URL>" [options]
```

**Options:**

- `--output <path>`: Directory where the `.md` file will be saved.
- `--attachments <name>`: Custom name for the folder containing images (default: `attachments`).

**URL Support:**

- **Sina Mobile**: `https://blog.sina.cn/dpool/blog/s/blog_xxx.html`
- **Sina Desktop**: `https://blog.sina.com.cn/s/blog_xxx.html` (Auto-converted to mobile for better parsing).

---

### 2. User Collection Listing (`user-collection`)

Fetches a complete list of articles from a user profile and generates a Markdown table for screening.

**Command:**

```bash
opencli sinablog user-collection "<User-ID>" --output list.md
```

- **Key Feature**: The `--output` flag ensures the Markdown table is saved properly for `batch-save`.

---

### 3. Batch Archiving (`batch-save`)

Processes a Markdown list generated by `user-collection` and archives all marked items.

**Command:**

```bash
opencli sinablog batch-save list.md [options]
```

- **Options**: Supports the same options as `save` (`--output`, `--attachments`).
- **Workflow**:

  1. Generate list via `user-collection --output list.md`.
  2. Mark desired articles:
     - **采集**：在 `collect` 列填入 `y`, `x`, `1` 或任何非 `0` 的字符。
     - **已采集/忽略**：填入 `0` 表示该项已通过其他方式处理。
     - **待处理**：保持为空。
  3. Run `batch-save list.md`.
  4. **自动回写**：`batch-save` 执行成功后，会自动在 `list.md` 的 `archive` 列填入 `[[文件名]]` 双链，并将 `collect` 标记重置为 `0`（已采集状态），方便你追踪采集进度。

---

## Technical Requirements & Best Practices

- **URL Support**: Supports both browser URLs (desktop and mobile).
- **Strategy**: Uses a browser-based approach for high-quality content extraction and local media handling.
- **Obsidian Compatibility**: The `save` command is fully optimized for Obsidian vaults with relative attachment paths.
